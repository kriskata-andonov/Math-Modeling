# Транспортна Задача

Това е цялостно решение на транспортната задача, разработено на езика Julia. Приложението е изцяло конзолно (CLI) и поддържа автоматично балансиране, намиране на начален опорен план и оптимизация чрез Метод на потенциалите (MODI).

## 🌟 Възможности
- **Автоматично балансиране:** Поддържа затворени и отворени задачи (добавяне на фиктивни възли).
- **Начален опорен план:** Северозападен ъгъл или Минимален елемент (Least Cost).
- **Оптимизация:** Метод на потенциалите (MODI) с автоматично откриване на цикли.
- **Интерактивност:** Лесно въвеждане на данни директно в конзолата стъпка по стъпка.

## 🛠 Инсталация и Изпълнение

За да стартирате приложението на която и да е операционна система, първо трябва да имате инсталирана **Julia**. Можете да я изтеглите от [официалния сайт](https://julialang.org/downloads/).

Тъй като това е чист Julia скрипт без външни зависимости, **не се изисква** инсталиране на допълнителни пакети (няма нужда от `Pkg.add`).

### 🐧 Linux
1. Отворете вашия терминал.
2. Клонирайте или изтеглете хранилището локално:
   ```bash
   git clone https://github.com/kriskata-andonov/Math-Modeling.git
   cd Math-Modeling
   ```
3. Стартирайте програмата:
   ```bash
   julia src/TransportationSolver.jl
   ```

### 🍎 MacOS
1. Отворете приложението **Terminal** (чрез Spotlight: `Cmd + Space` -> напишете "Terminal").
2. Клонирайте или изтеглете хранилището локално:
   ```bash
   git clone https://github.com/kriskata-andonov/Math-Modeling.git
   cd Math-Modeling
   ```
3. Изпълнете програмата:
   ```bash
   julia src/TransportationSolver.jl
   ```

### 🪟 Windows
1. Свалете проекта (например като ZIP файл от GitHub чрез бутона "Code" -> "Download ZIP") и го разархивирайте в удобна за вас папка.
2. Отворете разархивираната папка `Math-Modeling` в File Explorer.
3. Кликнете с десен бутон в празно пространство вътре в папката (ако сте на Windows 10, задръжте клавиша `Shift` и кликнете с десен бутон) и изберете **"Open PowerShell window here"** (или "Open in Terminal", ако сте на Windows 11).
4. В черния/синия прозорец напишете следната команда и натиснете `Enter`:
   ```powershell
   julia src/TransportationSolver.jl
   ```

## 📂 Структура на проекта
- `src/TransportationSolver.jl` - Главен входен файл и интерактивно меню.
- `src/Models.jl` - Структури на данните (`TransportData`, `Solution`).
- `src/InputHandler.jl` - Обработка на входа от потребителя и логика за балансиране.
- `src/InitialPlan.jl` - Алгоритми за намиране на начален план.
- `src/Optimizer.jl` - Метод на потенциалите (MODI) за оптимизация.
- `src/Utils.jl` - Помощни функции за принтиране на красиви таблици.
